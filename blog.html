Blog - 01. What is DOM and what is the purpose of DOM?

The Document Object Model (DOM) is a programming interface for web documents. The DOM represents the document as nodes and objects that way, programming languages can interact with the page.

With the Document Object Model, programmers can create and build documents, navigate their structure and add, modify, or delete elements and content. Anything found in an HTML or XML document can be accessed, changed, deleted, or added using the Document Object Model.

Blog - 03. What is event bubble?

When an event is in an element inside another element, and both elements have registered a handle to that event. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.

Event bubbling is a method of event propagation in the HTML DOM API.

Syntax:
addEventListener(type, listener, useCapture)

Blog - 04. What is a callback function and why will you use it?

A callback function is one of the superpowers of JavaScript. It is the way JavaScript passes a function into another function as an argument. Arguments in JavaScript are values that are passed to the parameters of a function. For example: showing a pop-up message welcoming a user after the user inputs a username.

We use callback() function's primary intent is to run code to any action or event in response. Which events can be user defined, such as mouse hovering, clicking or any button. When we use a callback() function, we can instruct their application like "Click on key to run the program."
